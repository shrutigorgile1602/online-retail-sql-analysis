# 📊 Online Retail Analytics & Customer Behavior Insights using SQL

---

## 📌 Project Overview
This project focuses on analyzing transactional data from an online retail platform using SQL to generate meaningful business insights. The analysis explores customer purchasing patterns, product performance, seller contribution, and delivery efficiency.

The objective is to transform raw structured data into actionable insights that can support strategic decision-making and improve business operations.

---

## 🎯 Business Objective
- Understand customer buying behavior and trends  
- Analyze product category performance  
- Evaluate seller contribution to revenue  
- Identify operational inefficiencies in delivery  
- Support data-driven business decisions  

---

## 🛠️ Tools & Technologies
- SQL Server  
- SQL Server Management Studio (SSMS)  
- CSV Dataset  

---

## ⚙️ Setup Instructions
1. Install SQL Server and SSMS  
2. Create a new database  
3. Create tables using the provided schema  
4. Import dataset CSV files into tables  
5. Execute SQL queries for analysis  

---

## 🗂️ Dataset Overview
The dataset includes:
- Customers  
- Orders  
- Products  
- Sellers  
- Payments  
- Reviews  

These tables are interconnected to provide a complete view of the e-commerce ecosystem.

---

- Identify most frequently sold products

## 📈 Key Insights
- Seller distribution is geographically diverse across multiple regions  
- Order volume indicates strong platform growth over time  
- Certain product categories contribute significantly to total revenue  
- Major cities act as high-demand centers  
- Digital payment methods (especially credit cards) dominate transactions  
- Top-selling categories drive a large portion of total sales  
- Delivery delays highlight opportunities for operational improvement  
- Customer satisfaction levels are generally positive across categories
